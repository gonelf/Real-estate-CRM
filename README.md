# NO-CRM | Real Estate

A simple real estate CRM for managing properties and leads. Data is stored locally in the browser using localStorage.

## Features

- **Properties** - Add, edit, and delete properties with address, status (Available / CPCV / Sold), and optional photos
- **Leads** - Track leads per property with name, email (optional), phone (optional), and status (Good / Maybe / Bad)
- **Comments** - Add free-form comments to any lead

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- localStorage for persistence

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
